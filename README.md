Classificador de Nível de Herói

Projeto desenvolvido como parte do desafio da DIO.

Tecnologias
- JavaScript

Conceitos aplicados
- Variáveis
- Operadores
- Estruturas de decisão

Como executar
1. Instale o Node.js
2. Execute o comando:
   node index.js

Resultado
O sistema classifica o herói de acordo com sua experiência (XP).
